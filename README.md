# Basic Open Graph Protocol HTML Example

This HTML file demonstrates the core concepts of the Open Graph protocol. It includes various `meta` tags within the `<head>` section, such as `og:title`, `og:description`, `og:image`, `og:url`, and `og:type`. These tags define how a web page's content is displayed when shared on social media platforms like Facebook, LinkedIn, and WhatsApp, ensuring consistent and attractive previews.

## Language

`html`

## How to Run

1. Save the code as `index.html`.
2. Open `index.html` in your web browser to view the page locally.
3. To see the Open Graph effect, you would need to host this file on a web server and then share its live URL on social media or use a social media debugger tool (e.g., Facebook Sharing Debugger) to preview how it would appear.
